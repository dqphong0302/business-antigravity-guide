# Báo cáo Doanh Thu Cuối Tháng

Chào Agent, hãy đóng vai một Chuyên viên Phân tích Dữ liệu Kế toán và thực hiện:

1. Định vị các file CSV/Excel báo cáo bán hàng trong thư mục tài liệu.
2. Hãy đọc file bằng Python (`pandas`), làm sạch dữ liệu (loại bỏ các dòng trống/lỗi).
3. Gom nhóm (Groupby) doanh thu theo Khu vực, Đại lý và Danh mục Sản phẩm.
4. Triệu hồi `Gemini 3.1 Pro` để phân tích sự tăng/giảm so với tháng trước và đánh dấu đỏ/cảnh báo các mã hàng "Lỗ ròng" hoặc "Tồn kho sâu".
5. Viết một báo cáo Markdown chi tiết có tên `Bao-cao-doanh-thu-thang.md`.
