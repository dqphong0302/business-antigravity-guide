# Soi Biểu Đồ VN30 Mật Phục Điểm Mua

```bash
Nhập Lệnh Này Vào Khung Chat:
```

> Kích hoạt `/loc-co-phieu`
>
> 1. Gọi Skill `loc_co_phieu`.
> 2. Kéo API Data giá trực tiếp và Cây Khối lượng từ **bảng giá SSI iBoard** cho danh sách 30 mã VN30.
> 3. Lọc cổ phiếu theo **Phân Tích Kỹ Thuật (TA):** Đo RSI, Phân kỳ MACD, Test Đáy Cũ (Hỗ Trợ).
> 4. Cho Browser Agent đọc lướt News Thị trường 24h để chặn tín hiệu nổ lệnh vì Tin dỏm.
> 5. Xuất Khuyến nghị "NGƯỠNG VÀO TIỀN" (Entry: X, Stoploss: Y) kèm giải trình Biểu Đồ lý do nên mai phục cổ phiếu này.
